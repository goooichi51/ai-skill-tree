# AIエージェント入門

自律的にタスクを実行するAIエージェントの概念と活用方法を学びます。

## 概要

| 項目 | 内容 |
|------|------|
| 難易度 | 🟠 Lv3 応用 |
| 所要時間 | 4-6時間 |
| 前提知識 | [[chat-ai/prompt-engineering\|プロンプトエンジニアリング]] |

## AIエージェントとは

従来のAI（チャット）との違い：

| 項目 | チャットAI | AIエージェント |
|------|-----------|---------------|
| 動作 | 1問1答 | 複数ステップを自律実行 |
| 判断 | 人間が指示 | AI自身が判断 |
| ツール | なし | ブラウザ、ファイル操作等 |
| 例 | ChatGPT | Claude Computer Use |

## 代表的なAIエージェント

### Claude Computer Use
- Claudeがコンピューターを操作
- ブラウザ操作、ファイル編集など
- 2024年後半に登場

### GPTs + Actions
- ChatGPT上でカスタムエージェント
- 外部APIと連携
- ノーコードで作成可能

### AutoGPT / AgentGPT
- 目標を与えると自律的にタスク分解・実行
- オープンソース

## 活用シーン

- リサーチの自動化
- データ収集・整理
- レポート作成
- 複雑なワークフローの自動実行
