# 画像生成入門

テキストから画像を生成するAIの基礎を学びます。

## 概要

| 項目 | 内容 |
|------|------|
| 難易度 | 🟢 Lv1 入門 |
| 所要時間 | 1時間 |
| 前提知識 | なし |
| 主なツール | DALL-E, Midjourney, Stable Diffusion |

## 学習内容

### 画像生成AIとは
- テキストプロンプトから画像を生成する技術
- 2022年以降急速に進化
- ビジネスでの活用が広がっている

### 主要サービス比較

| サービス | 特徴 | 価格 |
|----------|------|------|
| DALL-E | ChatGPT内で使える、手軽 | 有料（ChatGPT Plus） |
| Midjourney | 高品質、アート向け | 月額$10〜 |
| Stable Diffusion | 無料、カスタマイズ性高い | 無料〜 |

### 基本的なプロンプト

```
シンプルな例：
「オフィスで働くビジネスマン、明るい雰囲気」

詳細な例：
「モダンなオフィス、ノートPCで作業する30代日本人男性、
 窓から自然光、ミニマルデザイン、4K、フォトリアリスティック」
```
