# 🎨 生成AI

画像・動画・音声などのコンテンツを生成するAIスキル群です。
クリエイティブな業務を効率化し、表現の幅を広げます。

## スキル一覧

### 画像生成
- [[image-generation|画像生成入門]] - AI画像生成の基礎
- Midjourney活用（準備中）
- DALL-E活用（準備中）
- Stable Diffusion（準備中）

### 動画・音声（準備中）
- 動画生成入門
- 音声生成・文字起こし

## レベル別ロードマップ

```
Lv1 → DALL-Eで簡単な画像を生成
Lv2 → プロンプトで画風・構図を指定
Lv3 → Midjourneyで高品質な画像を生成
Lv4 → LoRA学習でスタイルをカスタマイズ
Lv5 → Stable Diffusion APIで自動生成システム
```

## 用途例

| 用途 | ツール | レベル |
|------|--------|--------|
| SNS投稿画像 | DALL-E, Canva | Lv1-2 |
| プレゼン資料 | Midjourney | Lv2-3 |
| 広告クリエイティブ | Midjourney, SD | Lv3-4 |
| ブランドアセット | カスタムLoRA | Lv4-5 |
