# N8N入門

オープンソースのワークフロー自動化ツール「n8n」の基礎を学びます。

## 概要

| 項目 | 内容 |
|------|------|
| 難易度 | 🟡 Lv2 基礎 |
| 所要時間 | 3-4時間 |
| 前提知識 | [[chat-ai/chatgpt-basics\|ChatGPT基礎]] あると良い |
| 公式サイト | [n8n.io](https://n8n.io) |

## n8nとは

- **オープンソース**の自動化ツール
- **400以上**のアプリ・サービスと連携
- **セルフホスト可能**でデータを自社管理
- **AI連携**が充実（OpenAI, Claude等）

## Zapierとの比較

| 項目 | n8n | Zapier |
|------|-----|--------|
| 価格 | 無料〜 | 有料（$20/月〜） |
| ホスティング | 自社/クラウド | クラウドのみ |
| カスタマイズ | 高い | 限定的 |
| 学習コスト | やや高め | 低め |

## 基本的なワークフロー例

```
[Webhook受信] → [ChatGPTで処理] → [Slackに通知]

[毎日9時] → [RSSフィード取得] → [要約生成] → [メール送信]
```

## 学習ステップ

1. n8n クラウドでアカウント作成
2. 基本的なワークフローを作成
3. AIノード（OpenAI）を追加
4. 本格的な業務自動化へ
