# ⚙️ 自動化

定型業務を自動化し、AIと組み合わせてワークフローを構築するスキル群です。
コードを書かずに複雑な自動化を実現できます。

## スキル一覧

### ノーコード自動化
- [[n8n|N8N入門]] - オープンソースの自動化ツール
- Zapier活用（準備中）
- Make（Integromat）活用（準備中）

### 簡易自動化（準備中）
- IFTTT入門
- Google Apps Script

## レベル別ロードマップ

```
Lv1 → IFTTTで簡単な自動化を体験
Lv2 → Zapierで複数アプリを連携
Lv3 → N8N/Makeで複雑なワークフロー構築
Lv4 → AI（ChatGPT API）をワークフローに組み込む
Lv5 → カスタムノード開発・大規模自動化
```

## 用途例

| 用途 | ツール | レベル |
|------|--------|--------|
| SNS投稿の自動化 | IFTTT, Zapier | Lv1-2 |
| メール自動返信 | Zapier + ChatGPT | Lv2-3 |
| データ収集・整理 | N8N | Lv3-4 |
| 営業プロセス自動化 | N8N + API連携 | Lv4-5 |
