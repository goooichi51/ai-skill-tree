# 🤖 対話型AI

テキストベースで対話しながらAIを活用するスキル群です。
ChatGPT、Claude、Geminiなどの大規模言語モデル（LLM）を使いこなすための知識を学べます。

## スキル一覧

### 基礎スキル
- [[prompt-engineering|プロンプトエンジニアリング]] - すべてのAI活用の土台
- [[chatgpt-basics|ChatGPT基礎]] - 最も普及している対話型AI
- [[claude-basics|Claude基礎]] - 長文処理に強いAI

### 発展スキル（準備中）
- Gemini活用
- Perplexity活用
- カスタムGPT作成

## レベル別ロードマップ

```
Lv1 → ChatGPT/Claudeで質問してみる
Lv2 → プロンプトを工夫して精度を上げる
Lv3 → 業務に合わせたプロンプトを設計
Lv4 → API連携で自動化
Lv5 → カスタムアプリ開発
```
