# プロンプトエンジニアリング

AIに対して効果的な指示を出すための技術。すべてのAI活用の土台となる最も重要な基礎スキルです。

## 概要

| 項目 | 内容 |
|------|------|
| 難易度 | 🟢 Lv1-2 |
| 所要時間 | 2-4時間 |
| 前提知識 | なし |

## 学習内容

### 基本テクニック（Lv1）
- 明確で具体的な指示の書き方
- コンテキストの提供方法
- 出力形式の指定

### 応用テクニック（Lv2）
- Few-shot プロンプティング
- Chain-of-Thought（思考の連鎖）
- ロールプレイング指示
