# 💻 開発・コーディング

AIを活用してコーディングを効率化し、独自のAIアプリケーションを開発するスキル群です。

## スキル一覧

### AIコーディング支援
- [[cursor|Cursor入門]] - AI搭載エディタ
- GitHub Copilot活用（準備中）
- Claude Code活用（準備中）

### API連携（準備中）
- OpenAI API入門
- Claude API入門
- LangChain入門

## レベル別ロードマップ

```
Lv1 → Cursorでコード補完を体験
Lv2 → AIにコードの説明・修正を依頼
Lv3 → GitHub Copilotで本格的に開発
Lv4 → OpenAI APIでカスタムアプリ開発
Lv5 → LangChain/RAGで高度なAIシステム構築
```

## 前提スキル

> [!note] プログラミング知識について
> Lv1-2は非エンジニアでも活用可能です。
> Lv3以上はプログラミングの基礎知識があると効果的です。
