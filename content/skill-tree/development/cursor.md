# Cursor入門

AI搭載コードエディタ「Cursor」の基礎を学びます。

## 概要

| 項目 | 内容 |
|------|------|
| 難易度 | 🟡 Lv2 基礎 |
| 所要時間 | 2時間 |
| 前提知識 | プログラミング基礎（なくても可） |
| 公式サイト | [cursor.com](https://cursor.com) |

## Cursorとは

- **VS Codeベース**のAI搭載エディタ
- **Claude / GPT-4**が組み込まれている
- **コード補完、チャット、編集**をAIが支援
- **無料プラン**でも十分に使える

## 主な機能

### 1. Tab補完
コードを書いている途中でAIが続きを予測・補完

### 2. Cmd+K（編集）
選択範囲に対して「〜に変更して」と指示

### 3. Cmd+L（チャット）
コードについて質問、新しいコードを生成

### 4. Composer
複数ファイルをまたいだ大規模な変更

## 非エンジニアでも使える

> [!tip] プログラミング初心者にもおすすめ
> 「こういうことがしたい」と日本語で伝えれば
> AIがコードを生成してくれます。
> 学習ツールとしても最適です。
